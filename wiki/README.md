# ReanimateMC Wiki

This directory contains a static HTML wiki website for the ReanimateMC Minecraft plugin.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth section switching with URL hash support
- **Search Functionality**: Search through different wiki sections
- **Copy-to-Clipboard**: Click on code snippets to copy them
- **Keyboard Navigation**: Use Alt + Arrow keys to navigate between sections
- **Back to Top**: Smooth scrolling back to top button
- **Mobile Friendly**: Collapsible sidebar for mobile devices

## Structure

- `index.html` - Main wiki page with all content
- `css/style.css` - Stylesheet with responsive design
- `js/script.js` - JavaScript for interactivity and navigation
- `images/` - Directory for any local images (currently using external URLs)

## Usage

1. Open `index.html` in any modern web browser
2. Navigate using the sidebar menu
3. Use the search box to quickly find sections
4. Click on code snippets to copy them to clipboard
5. Use Alt + Arrow keys for keyboard navigation

The wiki includes comprehensive documentation for:

- Command reference

## Customization

To customize the wiki:

1. Edit `index.html` to modify content
2. Adjust `css/style.css` for styling changes
3. Extend `js/script.js` for additional functionality
4. Add images to the `images/` directory as needed
